📊 Student AI Tool Usage Dataset Analysis

This project explores how students use AI tools such as ChatGPT, Copilot, Gemini, Claude, and DeepSeek for academic activities.
The dataset includes information about weekly usage hours, purpose of use, satisfaction levels, and academic performance.

The goal of this project is to demonstrate a complete data analysis workflow, including:
   Data Cleaning
     
   Exploratory Data Analysis (EDA)
     
   Data Visualization
     
   Machine Learning modeling

Through this analysis, we aim to understand AI adoption trends among students and explore
how AI tool usage may relate to learning habits and academic outcomes.

This project is ideal for data science learning, portfolio demonstration, and educational research on AI usage.
